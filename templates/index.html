Window + Shift + S로 캡처하고 이름, 학년, 반, 번호를 입력 후
Ctrl + V로 붙여넣고 제출하기 버튼을 클릭하세요.(여러 개 입력 가능)

중국어: 按下 Window + Shift + S 截图，然后输入姓名、年级、班级和号码，按 Ctrl+V 粘贴图片并点击“提交”按钮。（可多次提交）
러시아어: Нажмите Window + Shift + S для создания снимка экрана, затем введите имя, класс и номер, вставьте изображение через Ctrl+V и нажмите кнопку "제출하기". (Можно отправить несколько раз)
